# Распределённое приложение для параллельного решения СЛАУ методом ортогонализации

## Описание проекта

Приложение реализует распределённое решение системы линейных алгебраических уравнений (СЛАУ) в локальной вычислительной сети (ЛВС) средствами протокола TCP.

В качестве основного метода используется метод ортогонализации (Modified Gram-Schmidt, MGS).  
Для сравнения реализован последовательный метод Гаусса.

## Возможности

- решение СЛАУ методом Гаусса (последовательно);
- решение СЛАУ методом ортогонализации MGS (последовательно);
- распределённое решение MGS по схеме master-worker;
- обмен данными между узлами по TCP;
- чтение входных данных из файлов:
  - матрица системы;
  - вектор свободных членов;
  - список вычислительных узлов;
- нагрузочное тестирование и сравнение производительности;
- модульные тесты.

## Структура проекта

- `master.py` — управляющий узел (master);
- `worker.py` — вычислительный узел (worker);
- `gauss_solver.py` — последовательный метод Гаусса;
- `orth_solver.py` — последовательный метод ортогонализации (MGS);
- `io_utils.py` — загрузка матрицы, вектора и списка узлов;
- `data_generator.py` — генератор тестовых данных;
- `benchmark.py` — нагрузочное тестирование;
- `tests/` — модульные тесты.

## Запуск

### 1. Установить зависимости

```bash
pip install numpy pytest
```
### 2. Запустить worker-узлы

```bash
python worker.py --port 5001
python worker.py --port 5002
```

### 3. Запустить master

```bash
python master.py
```

 4. Запустить benchmark

```bash
python benchmark.py
```
### 5. Запустить тесты

```bash
python -m pytest
```
### Формат входных файлов

matrix.txt

Первая строка — размерность n, далее n строк по n чисел:
```bash
3
4 1 2
1 5 1
2 1 6
```

vector.txt

Первая строка — размерность n, далее n чисел:
```bash
3
7
8
9
```

nodes.txt 

Список узлов в формате host:port:
```bash
127.0.0.1:5001
127.0.0.1:5002
```
